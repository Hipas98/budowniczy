# Kalkulator JDG 🧮

Porównanie form opodatkowania dla jednoosobowej działalności gospodarczej w Polsce (2026).

Wpisujesz miesięczny przychód → dostajesz porównanie ryczałtu 8.5% / 12%, liniowego 19% i skali 12%/32% — ile zostaje netto przy każdej formie.

## Demo

> W budowie — wkrótce live

## Jak uruchomić lokalnie

```bash
cd projekty/kalkulator-jdg
pip install -r requirements.txt
streamlit run app.py
```

Python · Streamlit
